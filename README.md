# 🍽️ Bella Cucina — AI Reservation & Waitlist System

A demo of an AI-powered restaurant reservation and waitlist management system. Built as a single HTML file — no backend, no framework, no install needed.

---

## 🔗 Live Demo

👉 **[View Live Demo](https://exquisite-fox-ec2a07.netlify.app)**

---

## 💡 What It Does

### Guest View (left panel)
- Chat with an AI host to book a table
- AI asks for party size, name and phone number
- Handles natural language — guests can type however they want
- Joins the waitlist and gets a position + estimated wait time

### Staff View (right panel)
- Live waitlist updates in real time as guests book
- Seat, notify, or remove parties with one click
- SMS notification button (UI demo — Twilio integration ready)
- Shows free tables, parties waiting, and avg wait time

---

## 🚀 How to Run

Just open the file in any browser:

```bash
# Option 1 — double click
bella-cucina.html

# Option 2 — VS Code Live Server
Right click → Open with Live Server
```

No npm install. No server. No dependencies.
